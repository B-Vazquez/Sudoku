# Sudoku

This program takes in a text file containing a Sudoku puzzle consisting of numbers (0-9) and letters (a-f). The program then recursively finds every valid solution to the given puzzle.

The program currently assumes the input file follows the expected formatting (please view the example puzzle and solution). Any formatting errors in the input file will cause issues.

I plan to update the project to properly validate the provided input to ensure no bugs/crashes are experienced.

If running the program within VSCode via the .sln, ensure the input file is in the Project3-Sudoku folder (initially has a copy of the example input file which can be moved if needed), otherwise the file needs to be moved within the directory listed in the error when the code is ran.
